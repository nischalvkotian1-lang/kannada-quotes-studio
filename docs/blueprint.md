# **App Name**: Kannada Quotes Studio

## Core Features:

- Categorized Quote Browsing: Browse a rich collection of Kannada quotes, intuitively organized into various categories.
- Quote Interaction: Effortlessly share selected quotes to other applications or copy them to the clipboard.
- Favorite Quotes Management: Save preferred quotes to a personal favorites list for quick access and revisit anytime.
- AI-Powered Status Image Creator: Generate visually stunning status images from quotes, with an AI tool providing smart suggestions for background aesthetics, complementary colors, and elegant text layouts.
- Monetization through Ads: Integrate AdMob banner placeholders across the application to support ongoing development and maintenance.

## Style Guidelines:

- Color scheme: Dark theme, featuring a bold and energetic orange palette for a modern, sleek appearance. Primary interactive elements will use vibrant orange for emphasis.
- Primary color: HSL(30, 100%, 55%) or #FF9500, a dynamic and modern orange for key interactive components.
- Background color: HSL(30, 10%, 8%) or #2C2A27, a very dark, subtly warm charcoal grey, providing a deep canvas for the vibrant primary and accent colors.
- Accent color: HSL(0, 70%, 40%) or #AD1F1F, a rich, deep red to provide complementary contrast and highlight secondary actions.
- Main UI font: 'Inter' (sans-serif), for its modern, clean, and objective aesthetic across UI elements and Latin text.
- Use minimalist, flat line-art icons that are easily discernible against the dark background, utilizing the primary orange for active states and critical actions.
- A contemporary card-based layout for quotes, providing a clear visual hierarchy and generous negative space. Features a prominent and persistent bottom navigation for easy access to main app sections.
- Implement smooth and subtle animations for screen transitions, category changes, and interactive feedback, contributing to a fluid and responsive user experience without distraction.
